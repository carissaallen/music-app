Work

Discover new music based on what you enjoy.

To explore and strengthen our skills in full-stack development, we brainstormed a web application that incorporates a neat third-party API. Together, we built an application that will generate a music playlist based on the user's favorite song, artist, or genre. We loved this idea because it's something we would use ourselves, and can expand on in the future.

Spotify Web API

The Spotify Web API provides music recommendations based on a "seed" artist song, or genre. To use these endpoints, we obtained authorization for our application using Spotify's Authorization Code Flow. This allowed us to obtain an access token that could be refreshed after a successful user login.
